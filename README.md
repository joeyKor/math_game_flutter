# 🧠 수학 게임 (Math Game Flutter)

모던하고 세련된 UI를 갖춘 다채로운 수학 퍼즐 게임 모음집입니다.

## 🎮 포함된 게임

### 1. 🏹 수식 양궁 (Math Archery)
- 주어진 숫자와 연산자를 조합하여 목표 숫자를 맞추는 지능형 퍼즐입니다.
- **특징**: 
  - 각 과녁마다 다른 점수 (1pt, 3pt, 10pt, 50pt) 부여
  - **콤보 시스템**: 연속 적중 시 보너스 점수 지급 (5/10/15 콤보) 및 화려한 연출
  - 'PASS' 기능 및 'Skip Round' 기능 제공 (각 1pt 감점)
  - 화려한 화살 발사 애니메이션 및 물리 기반 타격 효과
  - **안정성**: 윈도우 환경에서의 프리징 현상 해결 및 이중 안전 장치 적용

### 2. ⚡ 수식 암산 (Flash Mental)
- 화면에 빠르게 지나가는 숫자들을 기억하고 합계를 맞추는 게임입니다.
- **특징**:
  - 10문제 연속 도전 모드
  - 오답 시 즉시 종료되는 고난도 모드
  - 실시간 타이머 및 최고 기록 갱신

### 3. 🔢 사칙연산 퀴즈 (Square Quiz)
- 사각형 형태의 레이아웃에서 빠르게 사칙연산 문제를 푸는 게임입니다.
- **특징**:
  - 입체적인 파티클 효과와 타격감 있는 UI

### 4. 🔍 소수 찾기 (Prime Detector)
- 4x4 그리드에서 소수(Prime Number)를 찾아내는 게임입니다.
- **특징**:
  - **난이도 조절**: Level 1 (11-999), Level 2 (1001-9999) 지원
  - 소수 분해 안내 메시지 제공
  - **4 스트라이크 제도**: 4번 틀릴 시 게임 종료

### 5. 📅 요일 방정식 (Weekday Equation)
- 특정 날짜의 요일을 맞추거나 요일 간의 관계를 계산하는 게임입니다.
- **특징**:
  - 중복 문제 방지 로직 적용
  - 난이도별 최적화된 날짜 범위 제공
  - 콤팩트한 4x3 키패드 레이아웃 적용

### 6. 🛡️ 분수 배틀 (Fraction Battle)
- 두 분수의 크기를 비교하거나 덧셈을 수행하는 게임입니다.
- **특징**:
  - **기약분수 필수**: 정답 입력 시 반드시 기약분수 형태로 입력해야 인정

### 7. 📘 수학 백과 (Math Encyclopedia)
- 수학적 개념과 공식을 시각적 다이어그램과 함께 제공합니다.

### 8. 팩토리얼 계산기 (Factorial Calculator)
- 큰 숫자의 팩토리얼을 계산하고 그 과정을 확인하는 도구입니다.

### 9. ⚖️ 크기 비교 (Sum Comparison)
- 양쪽 수식의 합 중 어느 쪽이 더 큰지 빠르게 판단하는 게임입니다.
- **특징**:
  - 공개 시간(Reveal Time) 중에도 빠른 선택 가능
  - 슬림하고 직관적인 타이머 바

## 🚀 주요 기능

- **통합 점수 시스템**: 모든 게임의 결과가 `UserProvider`를 통해 전역 포인트로 관리됩니다.
- **포인트 히스토리**: 상세한 포인트 획득/차감 내역을 확인하고 관리할 수 있습니다.
- **음성 및 진동 피드백**: TTS와 햅틱 피드백을 통해 몰입감 있는 게임 경험을 제공합니다.
- **프리미엄 UI/UX**: 다크 모드 기반의 세련된 디자인, 부드러운 애니메이션을 제공합니다.
- **로컬 데이터 보존**: `SharedPreferences`를 사용하여 사용자 데이터가 안전하게 저장됩니다.

## 🛠 기술 스택

- **Framework**: Flutter
- **Language**: Dart
- **State Management**: Provider
- **Storage**: SharedPreferences
- **Feedback**: Flutter TTS, Vibration
